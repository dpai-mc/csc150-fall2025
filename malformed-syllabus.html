Course Description

This course introduces students to Python programming within a cybersecurity context, emphasizing both defensive automation and offensive security awareness. Students learn to develop security tools, automate defensive processes, and understand attack methodologies to build comprehensive cybersecurity solutions. The course follows a "blue team with red team awareness" philosophy, teaching students to think like both defenders and attackers to create more effective security measures.

Through hands-on labs and real-world case studies from financial services, healthcare, and e-commerce sectors, students develop practical skills in vulnerability assessment, incident response automation, network security analysis, and web application security testing. All offensive techniques are taught within a controlled, ethical framework to improve defensive capabilities and prepare students for comprehensive cybersecurity careers.

    Course Description

    This course will provide students with Python programming skills essential for today's cybersecurity landscape. Students will master core Python concepts while applying them to automate security tasks, detect vulnerabilities, and prepare scripts/programs across the cybersecurity landscape. This is a hands-on class designed to enhance a student's understanding and proficiency in applying Python for cybersecurity solutions. Previous networking and cybersecurity knowledge is recommended.

    Student Learning Outcomes

    • Write Python programs that implement core programming concepts like variables, data types, control flow, functions, modules, file I/O, and exception handling.
    • Develop scripts for network reconnaissance and scanning which may include one or more of the following; port scanning, banner grabbing, service identification, and vulnerability detection.
    • Create scripts for penetration testing that implement exploit development, password cracking, and/or privilege escalation.
    • Create tools for incident response and forensics that use or perform log analysis, malware analysis, and/or evidence collection.
    • Describe the ethical and legal implications of using programming skills for cybersecurity purposes.

    Ethical Guidelines & Legal Considerations

    Throughout this course, emphasis is placed on ethical hacking principles and responsible use of cybersecurity skills. All students must agree to and follow these guidelines:

    • Ethical hacking principles and responsible disclosure
    • Legal boundaries in cybersecurity testing and tool development
    • Permission and authorization requirements for all security testing
    • Professional conduct in cybersecurity careers
    • Academic integrity in programming assignments

    Important: All tools and techniques taught in this course are for educational purposes and authorized testing environments only. Unauthorized use of these skills against systems you do not own or have explicit permission to test is illegal and grounds for immediate course failure and potential legal action.
